Table View Goodies
======================

Use `CustomGroupedCellBackground` to give table view cell any kind of background you want.

Rounded corners are handled in a `UITableView` category (instead of a subclass) to minimize headache. Just put any cell that is a subclass of `CustomGroupedCell` in your table view, and Voil√†!
